## 機能
### ゲストユーザー
- 投稿一覧確認
- タイトルクリックで投稿の詳細を確認
### ログインユーザー
- 投稿一覧確認
- タイトルクリックで投稿の詳細を確認
- 投稿の新規作成
- 投稿の編集
- 投稿の削除
- 他のユーザーの投稿にコメント投稿
- コメント編集
- コメント削除
- 投稿のいいね登録
- 投稿のいいね削除
- ユーザー名変更
- パスワード変更
- アカウント削除

## 課題
- いいね登録時にフラッシュメッセージが表示されない
